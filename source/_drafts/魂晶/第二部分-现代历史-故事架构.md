# 第一版

旧人类主角姐弟和朋友外出游玩，展现两人情感

意外使得两人离开安全区，进入禁区，遇到异种野兽，姐姐重伤

保守派新人类出现，弟弟A求救，犹豫是否要杀死两人

魂晶回归派系新人类到场，新人类混战

新人类两败俱伤，一个魂晶回归派系新人类垂死被抛弃，姐姐死亡

A被新人类转化部分灵魂，获得魂晶操控能力和新人类的魂晶

A带着姐姐尸体回到村庄，发现村庄被魂晶回归派新人类摧毁

A被保守派新人类发现，带回前线基地抚养，一直试图复活姐姐

保守派新人类向弟弟介绍历史，他和前线基地新人类建立情感

A能力日渐增强，越来越渴望复仇

他独自前往魂晶回归派系新人类据点，被控制住，押送面见雷希雅

前线基地的人们秘密组织解救A，被雷希雅发现，所有人被活捉

所有人聚集在雷希雅面前，雷希雅请求A杀死自己

A杀死雷希雅，雷希雅聚集的所有魂晶崩塌，混乱中前线基地的人们阻止了肆意杀戮的A，受到感化的A最后和前线基地的人们一起回到大家庭中

## 更多的说明

1. 几乎没有人能够在转化部分灵魂的情况下存活，只有历史上的伊西尔，雷希雅和上面的A。也因此，最初所有新人类都把A视为宝贵的资源。魂晶回归派系希望通过A的力量加速不可知世界和物质世界的交换，重启世界；保守派新人类希望保持当前的状态，阻止A毁灭世界。
2. 雷希雅虽然在转化部分灵魂后存活，却一直为自己发动红色风暴屠杀生灵极度自责，希望自我了断，但是自身的生命意志不允许自己死亡。所以她希望借助A杀死自己，所以魂晶回归派系新人类在寻找别的转化部分灵魂同时存活的人完成大业。
3. 位于前线基地的保守派新人类最初只是为了防止A毁灭世界，在一同生活中对A从同情转化为理解，认同，最终和A形成一个大家庭，让A回归了人性。
4. 在A和前线基地的人们回前线基地的时候，通过将死的魂晶回归派系新人类口述，表现出两个派系新人类观念的巨大不同，表现出人性的美好与自私短视，但同样对人性发出赞歌。
